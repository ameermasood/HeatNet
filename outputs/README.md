# Outputs

This folder holds generated artifacts from the unified task scripts.

Suggested usage:

- `outputs/checkpoints/`: trained model checkpoints
- `outputs/histories/`: training history JSON files
- `outputs/predictions/`: pose prediction JSON outputs
- `outputs/evaluations/`: evaluation summaries and metrics

Due to repository and GitHub size limits, the actual generated outputs are intended to remain local. Only lightweight placeholder files are committed here to preserve the folder structure.
